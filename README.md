VocalGuard Engine v1.0

An advanced, secure, and intelligent system-level core developed in Java. VocalGuard Engine combines dual-factor biometric voice frequency verification with real-time OS automation and web-service handling.

Key Features

Voice Biometric Authentication Core
Real-time frequency analysis utilizing Fast Fourier Transforms to process analog audio signals and verify user identity before system access.

Low-Level OS Automation
Seamless execution of native Windows applications like Notepad interacting directly with the system kernel via the Java Runtime environment.

Web Integration Platform
Cross-platform default browser dispatching and target URL routing using the native Java Desktop API.

Enterprise Automation Matrix
Pre-configured structural placeholders for continuous deployment of automated communication streams such as WhatsApp Automation via Selenium.

Setup and Installation

1. Clone the Repository
git clone https://github.com/khizaraliali41-hash/VocalGuard-Engine1.git

2. Environment Configuration
Open the project directory folder inside VS Code.
Verify that the library mapping in settings points correctly to the dependency path:

"java.project.referencedLibraries": [
    "lib/**/*.jar"
]

3. Execution
Compile and initialize the main operational engine thread via Main.java
